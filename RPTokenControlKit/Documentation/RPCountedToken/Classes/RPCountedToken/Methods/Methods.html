count

Abstract: getter for the ivar count 
- (int)count;

countCompare:

- (NSComparisonResult)countCompare:(RPCountedToken*)other;

Result: if count of the receiver < count of other, returns NSOrderedDescending. If count of the receiver > count of other, returns NSOrderedAscending. If count of the receiver == count of other, returns result of -textCompare:other.

ellipsisToken

Abstract: gets a special "ellipsis" RPCountedToken 
+ (RPCountedToken*)ellipsisToken;

The text of the special "ellipsis" RPCountedToken is a one-character string, the unicode eliipsis (0x2026). The count is 0. This special RPCountedToken is used by RPTokenControl as the last token when there are too many to fit in the given frame.

Result: an autoreleased instance of the special "ellipsis" RPCountedoken

isEllipsisToken

- (BOOL)isEllipsisToken;

With the current design, -[myCountedToken isEllipsisToken] is equivalent to -[myCountedToken count] == 0 && myCountedToken != nil. This method is a convenience and attempt to avert future bugs.

Result: If the receiver is an ellipsis token, returns YES. Otherwise, returns NO.
